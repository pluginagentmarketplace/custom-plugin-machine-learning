# /career-path

Get personalized career path recommendations based on your interests, experience, and goals.

## Description

The `/career-path` command provides tailored career guidance, including role progression, salary expectations, required skills, and learning timelines.

## Usage

```
/career-path
/career-path junior-to-senior
/career-path specialization <role>
/career-path salary-growth
/career-path remote-options
```
