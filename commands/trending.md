# /trending

Discover the hottest and most in-demand technologies to learn right now.

## Description

The `/trending` command shows you the most popular and trending technologies in the developer community, based on industry demand, job market trends, and learning patterns.

## Usage

```
/trending
/trending category <category>
/trending by-demand
/trending growth-rate
```
