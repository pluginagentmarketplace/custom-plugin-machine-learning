# /explore

Explore all available agents, specializations, and learning opportunities in the Developer Roadmap ML Plugin.

## Description

The `/explore` command lets you browse and discover all 8 specialized agents, their areas of expertise, and available learning paths. Perfect for understanding what each specialization covers.

## Usage

```
/explore
```

## What It Does

Displays comprehensive information about:

1. **All 8 Agents** with their specializations
2. **Key expertise areas** for each agent
3. **Available skills** (5-8 per agent)
4. **Real-world projects** for each specialization
5. **Learning recommendations**
