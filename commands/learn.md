# /learn

Start a personalized learning journey through the Developer Roadmap ML Plugin.

## Description

The `/learn` command helps you choose a learning path based on your goals and current skill level. It guides you through structured learning paths across 8 different specialization areas.

## Usage

```
/learn
```

## What It Does

1. **Assess Your Level**: Choose between Beginner, Intermediate, Advanced, or Expert
2. **Select Specialization**: Choose from 8 specialization areas:
   - Frontend Development
   - Backend Development
   - Full Stack & Web
   - Mobile & Cross-Platform
   - Database & Data Management
   - Cloud & DevOps
   - AI/ML & Data Science
   - Specialized Roles & Architecture

3. **Get Recommendations**: Receive a personalized learning path with:
   - Structured learning modules
   - Recommended skills to focus on
   - Project ideas at your level
   - Best practices and patterns
   - Time estimates for completion

## Tips

- Start with Beginner if you're new to programming
- Choose a specialization that interests you most
- Follow the recommended learning path
- Build projects as you learn
- Use `/explore` to learn about other specializations
- Use `/skills` to deep dive into specific skills
