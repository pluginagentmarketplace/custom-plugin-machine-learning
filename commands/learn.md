# /learn

**Start a personalized, AI-powered learning journey through the Developer Roadmap ML Plugin.**

## Description

The `/learn` command is your personal learning guide. It uses AI to understand your current level, goals, and interests to create a customized learning path with specific skills, projects, and milestones.

## Usage

```
/learn
/learn [specialization]
/learn [specialization] [level]
/learn status
/learn recommend
```

## Specializations

1. **Frontend Development** - Modern UI, React, performance, accessibility
2. **Backend Development** - APIs, databases, microservices, security
3. **Full Stack & Web** - Next.js, complete applications, monorepos
4. **Mobile & Cross-Platform** - React Native, Flutter, native iOS/Android
5. **Database & Data** - SQL, NoSQL, data modeling, optimization
6. **Cloud & DevOps** - Docker, Kubernetes, AWS, infrastructure
7. **AI/ML & Data Science** - Machine learning, deep learning, MLOps
8. **Specialized Roles** - System design, architecture, leadership

## Skill Levels

- **Beginner (0-1 years):** Foundations, setup, fundamentals
- **Intermediate (1-3 years):** Building production features
- **Advanced (3-5 years):** Optimization, patterns, complexity
- **Expert (5+ years):** Mastery, system design, leadership
