# /projects

Browse and discover hands-on projects organized by difficulty, technology, and skill level.

## Description

The `/projects` command provides curated real-world projects across all specializations. Build portfolio pieces while learning.

## Usage

```
/projects
/projects list <level>
/projects search <keyword>
/projects by-tech <technology>
/projects gallery
```
